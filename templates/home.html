Try Dashboard (login needed)

Decorator Quick Reference

@login_required

User must be logged in. If not, redirects to LOGIN_URL.

from django.contrib.auth.decorators import login_required

@login_required
def dashboard(request):
  return render(request, 'dashboard.html')
Try it
@user_passes_test

Pass a custom function that returns True/False.

def is_superuser(user):
  return user.is_superuser

@user_passes_test(is_superuser)
def admin_only(request):
  ...
Try it
@permission_required

Checks Django's Permission model. Format: 'app.codename'.

@permission_required('pages.delete_article',
  raise_exception=True)
def delete_article(request, pk):
  ...
@staff_member_required

User must have is_staff=True. From django.contrib.admin.

from django.contrib.admin.views.decorators \
  import staff_member_required

@staff_member_required
def staff_panel(request):
  ...
Try it
@require_POST / @require_http_methods

Restricts allowed HTTP methods. Returns 405 otherwise.

from django.views.decorators.http \
  import require_POST

@require_POST
def quick_publish(request):
  ...
Stacking Decorators

Combine multiple decorators. Applied bottom → up.

# 2nd: checks login
@login_required
# 1st: checks POST
@require_POST
def quick_publish(request):
  ...
